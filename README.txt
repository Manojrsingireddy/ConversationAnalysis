CONVERSATION ANALYSIS PROGRAM:

Description:
This program is used by mac users who would use iMessage from their macs. It can be run to conduct a variety of different interesting conversation
analysis tasks. See below list for currently implemented tasks and future tasks.

Implemented: 
1. Time Difference Analysis
2. Profanity Usage
3. Time of day most messages sent/received

In Progress:
1. Usage of particular words/Word Cloud Generation
2. Type of message composition (pie chart) audio, text, image, emote, etc.
3. Message length
4. UI Development

How to Use:
Clone Github repo to your Mac Computer OS (Big Sur or later). Enter System preferences and give your 
preferred editor full disk access. Open in the editor and run AnalysisRun.py.
